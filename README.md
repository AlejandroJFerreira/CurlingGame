# Multiplayer Curling Game

## Overview
This is a **real-time multiplayer curling game** built with **JavaScript, Node.js, HTML, and Socket.IO**. Players can join from different browser tabs and compete to place curling irons as close to the center as possible. The project highlights **client-server architecture, real-time networking, and object-oriented game design**.

## Tech Stack
- Languages: JavaScript  
- Server: Node.js  
- Networking: Socket.IO  
- Front-end: HTML/CSS  
- Tools: Command Prompt, Browser  
- Environment: Windows 11  
- Versions: Node v16.17.0, NPM 8.19.2  

## What I Built
- Developed a Node.js client-server system allowing multiple players to interact in real-time.  
- Implemented Socket.IO for low-latency communication between clients and server.  
- Designed modular game logic using object-oriented principles.  
- Built browser-based gameplay with interactive mechanics, including throwing curling irons and scoring.  

## How to Run
1. Clone the repository and navigate to the project folder.  
2. Install dependencies:  
   npm install  
   npm install socket.io  
3. Launch the server:  
   node server.js  
4. Open a browser and go to:  
   http://localhost:3000/chatClient.html  
5. Join the game as:  
   - Home  
   - Visitor  
   - Spectator (optional)  
6. Gameplay instructions:  
   - Click the center box, drag back, and release to throw the curling iron.  
   - Try to place your irons as close to the center as possible to score points.  
   - Use the "Leave" button to let another tab take your position if needed.
